## Description
- This is a weather iOS application
- Asks user for permission to track their location to provide local weather information 
- Shows weather hourly for the next 5 hours
- Shows weather daily for the next 5 days
- UI with Storyboard
- MVC Design Pattern

## Frameworks:
- Foundation
- UIKit
- CoreLocation
